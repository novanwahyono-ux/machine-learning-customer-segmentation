# Customer Segmentation using Machine Learning

A machine learning project focused on customer segmentation and classification using clustering algorithms and supervised learning techniques.

---

## Overview

This project aims to analyze customer-related data using machine learning approaches such as clustering, dimensionality reduction, and classification.

The workflow includes:
- data preprocessing
- clustering analysis
- PCA dimensionality reduction
- classification modeling
- model evaluation

The project is designed as a practical exploration of machine learning workflows and data-driven customer segmentation.

---

## Features

- Data preprocessing pipeline
- K-Means clustering implementation
- PCA dimensionality reduction
- Decision Tree classification
- Hyperparameter tuning
- Visualization and exploratory analysis
- Multiple trained ML models

---

## Tech Stack

- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
